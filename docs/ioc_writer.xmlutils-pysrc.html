 76      remove_namespace 
 77       
 78      Takes in a ElementTree object and namespace value.  The length of that  
 79      namespace value is removed from all Element nodes within the document. 
 80      This effectively removes the namespace from that document. 
 81       
 82      Returns the source document. 
 83      '''

 86      #print 'DEBUG: removing',ns 
 87      for elem in doc.getiterator():

 89              elem.tag = elem.tag[nsl:]
